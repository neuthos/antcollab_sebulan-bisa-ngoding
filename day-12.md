# Day 12

Selamat datang di Day-12 pembelajaran! Pada hari ini, kalian tidak perlu mempelajari hal baru. Namun, ini adalah kesempatan yang baik untuk mereview ulang materi-materi sebelumnya yang telah dipelajari sepanjang program "Sebulan Bisa Ngoding" dari Ant Collab.

Di hari-hari sebelumnya, Anda telah mempelajari berbagai konsep pemrograman, termasuk:

* HTML dan CSS: Membuat struktur halaman web dan mengatur tampilan menggunakan CSS.
* JavaScript Dasar: Mengenal variabel, kondisional, loop, dan fungsi dalam JavaScript.
* ES6: Memahami let, var, const, dan template literals dalam ES6.
* Array dan Nested Array: Menggunakan array dan nested array untuk menyimpan dan memanipulasi data.
* Function: Menciptakan blok kode yang dapat digunakan kembali dengan function.
* Modular Function: Memahami manfaat dan penggunaan modular function dalam mengorganisasi kode.

Sebagai langkah review, Anda dapat mencoba untuk menulis kode dari beberapa materi sebelumnya, menyelesaikan tantangan-tantangan atau latihan-latihan yang diberikan, atau membuat projek sederhana yang menggabungkan beberapa konsep yang telah dipelajari.
